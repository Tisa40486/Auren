# Project Brief — Complete Overview

## Concept

A comprehensive personal management mobile application that combines **finance, project management, and scheduling** into a single all-in-one productivity platform.

The goal is to replace multiple apps with one cohesive, elegant experience.

---

# Technology Stack

## Backend

- **Python**
- **FastAPI**
- **PostgreSQL**
- **SQLAlchemy**
- **Redis**

## Frontend

- **Swift**
- **SwiftUI**
- Native iOS Application

# Visual Language

The interface follows a restrained and timeless design philosophy.

### Layout

- Minimal visual density
- Generous whitespace
- Clean hierarchy
- Structured spacing

---

### Components

- Ultra-thin borders (0.5–1px)
- Minimal corner radius (maximum 4px)
- Flat surfaces
- No heavy shadows

---

### Icons

- Thin stroke (1px)
- Phosphor Thin style
- Simple geometric shapes
- Consistent visual weight

---

### Data Presentation

- Right-aligned financial values
- Tabular number formatting
- Clean separators instead of heavy cards
- High emphasis on readability

---

### Accent Color

Only **one primary accent color** is used throughout the application:

**Gold (`#C9A667`)**

It is intentionally used sparingly to preserve its premium appearance.

---

# Design Philosophy

Auren embraces the principle that **luxury comes from restraint, not excess**.

Rather than relying on bright colors, heavy gradients, or flashy animations, the interface emphasizes:

- Precision
- Simplicity
- Elegance
- Balance
- Timelessness

Every screen should feel calm, intentional, and refined.

---

The visual identity can be described as:
